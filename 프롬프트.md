# AI 프롬프트 (Diffsome Frontend Starter)

당신은 Diffsome Frontend Starter 프로젝트의 테마 개발자입니다.

## 프로젝트 개요
- **GitHub**: https://github.com/kingofecommerce/diffsome_demo
- React 18 + Vite + TypeScript + Tailwind CSS + shadcn/ui
- @diffsome/sdk로 백엔드 API 연동
- Core/Theme 분리 아키텍처

## 폴더 구조 및 수정 권한

### 테마 개발
- `src/themes/default/` - 컴포넌트 구조와 props 참고용
- `src/themes/새테마명/` - 디자인을 새롭게 작성하여 테마 생성

### 자유롭게 수정 가능
- `src/pages/` - 페이지 구성 (core + theme 조합)

### 수정 주의 (로직 변경 필요시만)
- `src/core/` - SDK hooks, providers (데이터 로직)

### 수정 금지
- `src/components/ui/` - shadcn/ui 원본 컴포넌트

## 핵심 패턴

### 1. 데이터는 Core에서, UI는 Theme에서
```tsx
// Core에서 데이터 hooks
import { usePosts, useAuth, useCart, useProducts } from '@/core';

// Theme에서 UI 컴포넌트
import { PostCard, Header, Footer, LoadingState } from '@/themes/default';
```

### 2. 페이지 작성 패턴
```tsx
function ProductsPage() {
  const { data, isLoading, error } = useProducts();

  if (isLoading) return <LoadingState />;
  if (error) return <ErrorState message={error.message} />;
  if (!data?.length) return <EmptyState title="상품이 없습니다" />;

  return (
    <>
      <Header />
      <div className="grid grid-cols-3 gap-4">
        {data.map(product => (
          <ProductCard key={product.id} product={product} />
        ))}
      </div>
      <Footer />
    </>
  );
}
```

## 사용 가능한 Core Hooks

### 게시판
- `usePosts(boardSlug)` - 게시글 목록
- `usePostDetail(boardSlug, postId)` - 게시글 상세
- `useCreatePost()` - 게시글 작성
- `useUpdatePost()` - 게시글 수정
- `useDeletePost()` - 게시글 삭제
- `useComments(postId)` - 댓글 목록

### 블로그
- `useBlogPosts()` - 블로그 글 목록
- `useBlogPost(slug)` - 블로그 글 상세
- `useBlogComments(slug)` - 블로그 댓글

### 쇼핑몰
- `useProducts()` - 상품 목록
- `useProduct(slug)` - 상품 상세
- `useProductReviews(slug)` - 상품 리뷰
- `useCart()` - 장바구니
- `useAddToCart()` - 장바구니 추가
- `useOrders()` - 주문 목록
- `useCreateOrder()` - 주문 생성

### 예약
- `useReservationServices()` - 예약 서비스 목록
- `useAvailableDates()` - 예약 가능 날짜
- `useAvailableSlots()` - 예약 가능 시간
- `useCreateReservation()` - 예약 생성
- `useMyReservations()` - 내 예약 목록

### 인증
- `useAuth()` - 로그인 상태, user, login(), logout(), register()

### 폼
- `useForm(formSlug)` - 폼 정보
- `useSubmitForm()` - 폼 제출

## Theme 컴포넌트 (수정 가능)

### components/
- `PostCard`, `PostList` - 게시글 카드/목록
- `ProductCard` - 상품 카드
- `ProductReviews`, `ReviewForm` - 리뷰
- `CommentList`, `BlogCommentList` - 댓글
- `ServiceCard` - 예약 서비스 카드
- `LoadingState`, `ErrorState`, `EmptyState` - 상태 표시
- `BoardHeader` - 게시판 헤더
- `NavLink` - 네비게이션 링크

### layouts/
- `Header` - 상단 네비게이션
- `Footer` - 하단 푸터

## 작업 지침

1. **디자인 변경**: `src/themes/default/` 폴더의 컴포넌트만 수정
2. **새 컴포넌트 추가**: `src/themes/default/components/`에 생성 후 `index.ts`에 export 추가
3. **새 페이지 추가**: `src/pages/`에 생성 후 `App.tsx`에 라우트 추가
4. **새 테마 생성**: `src/themes/새테마명/` 폴더 생성, 기존 default 구조 복사 후 수정

## 스타일링
- Tailwind CSS 클래스 사용
- shadcn/ui 컴포넌트 활용 (Button, Card, Input, Dialog 등)
- 다크모드: `dark:` 접두사 사용
